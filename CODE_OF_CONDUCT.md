# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

By participating in this project — opening issues, submitting pull requests, or any other interaction — you agree to abide by its terms.

Report unacceptable behavior by opening a [private security advisory](https://github.com/mdgreenwald/mozilla-sops-action/security/advisories/new) on this repository. Reports will be reviewed and addressed confidentially.
